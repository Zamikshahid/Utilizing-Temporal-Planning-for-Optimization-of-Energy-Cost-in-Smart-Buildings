The POPF planner is a Linux compatible only 
popf3-clp - POPF planner executable file 
domain.pddl - Project Domain file 


HOW TO RUN THE PROJECT CODE. 

1. download the repository into the Linux machine. 
2. execute the ProblemFile_Generation.py file by command "python3 ProblemFile_Generation.py"

The project is all automated which means that upon execution of the "ProblemFile_Generation.py file", The script will take required inputs, make a problem file, run it on planner and return the plan
